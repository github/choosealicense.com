{ Which of the following best describes your situation? }

  • I want it simple and permissive.

    The MIT License is a permissive license that is short and to the point. It lets people do anything they want with your code as long as they provide attribution back to you and don’t hold you liable.

    jQuery and Rails use the MIT License.

  • I’m concerned about patents.

    The Apache License is a permissive license similar to the MIT License, but also provides an express grant of patent rights from contributors to users.

    Apache, SVN, and NuGet use the Apache License.

  • I want people to "pay it forward"

    The GPL (V2 or V3) is a copyleft license: You allow A to see and alter the code, but if A gives (or sells) it to B, A must allow B to see and alter as well. V3 is similar to V2, but if A puts the code in a machine, then gives that machine to B, the machine must allow B to run altered versions of the code.

    Linux, Git, and WordPress use the GPL.

{ What if none of these work for me? }

  • My content isn’t code.

    Check out Creative Commons.

  • I want more choices.

    More licenses are available.

  • I don’t want to choose a license.

    You don’t have to.
