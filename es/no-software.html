Licencias No-software

Las licencias de software de codigo abierto tambien se pueden utilizar para trabajos sin software y son a menudo la mejor opcion, especialmente cuando las obras en cuestion pueden ser editado y versionado como fuente. Elige una licencia de codigo abierto.

Datos, media, etc.

CC0-1.0, CC-BY-4.0, and CC-BY-SA-4.0 son licencias abiertas usadas para material no-software que varian desde datasets a videos. Notar que Creative Commons no recomienda sus licencias para ser usadas para software or hardware.

Documentacion

Cualquier licencia de software de codigo abierto o licencia abierta para medios (ver anterior) tambien se aplica al software documentacion. Si utiliza diferentes licencias para su software y su documentacion, asegurese de especificar que los ejemplos de codigo fuente en el La documentacion tambien esta autorizada bajo la licencia de software.

Fonts

The SIL Open Font License 1.1 mantiene las fuentes abiertas, permitiendoles ser utilizados libremente en otras obras.

Proyectos mixtos

Si su proyecto contiene una mezcla de diferentes tipos de material, puede incluir multiples licencias, siempre y cuando sea explicito sobre cuales La licencia se aplica a cada parte del proyecto. Ver el aviso de licencia para este sitio como un ejemplo (fin de pagina, Licencia).
